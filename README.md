Análise Exploratória de Dados: Notas de Filmes
Este projeto consiste em uma análise exploratória de dados utilizando bibliotecas de manipulação e visualização da linguagem Python. O foco principal é entender a distribuição de notas de filmes e o faturamento de grandes produções cinematográficas.

🚀 Tecnologias Utilizadas
Python

Pandas: Manipulação e tratamento de dados.

Seaborn: Visualização de dados estatísticos.

Matplotlib: Criação de gráficos personalizados.

Jupyter Notebook: Ambiente de desenvolvimento da análise.

📊 O que foi analisado
Distribuição de Notas: Estudo da média e mediana das avaliações de filmes contidos na base de dados MovieLens.

Comportamento Estatístico: Uso de Histogramas e Boxplots para identificar tendências centrais e dispersão dos dados.

Dados do TMDB: Comparação de orçamentos (budget) e receitas (revenue) de filmes.

Linguagens Originais: Análise da predominância da língua inglesa em relação a outras línguas nos dados do TMDB 5000.
